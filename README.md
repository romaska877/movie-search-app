# movie-search-app


A responsive movie search application built with HTML, CSS and JavaScript using the OMDb API.

## Features

- Search movies by title
- Fetch live movie data from an API
- Display movie posters, titles, years and types
- Movie details modal
- Full movie plot and IMDb rating
- Favorites system
- Save favorites with LocalStorage
- Search results and favorites tabs
- Loading and error states
- Dark/light mode
- Responsive cinematic UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- OMDb API
- LocalStorage
- GitHub Pages
